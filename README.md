## Prerequisites

- [Bun](https://bun.com/)

**MacOS (using `brew`)**

```bash
brew install oven-sh/bun/bun
```

**Ubuntu/Debian**

```bash
curl -fsSL https://bun.com/install | bash
```

### Run development server

From the root of the project run:

```bash
bun start
```

The development server will be available on http://localhost:3000
